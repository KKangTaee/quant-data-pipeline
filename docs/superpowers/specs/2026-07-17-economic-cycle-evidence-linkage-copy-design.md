# Economic Cycle Evidence Linkage Copy Design

## 목적

사이클맵 옆 Evidence 패널에서 어떤 근거가 현재 위치에 사용되고 어떤 근거가 향후 1·2개월 경로에 추가되는지 사용자가 바로 구분할 수 있게 한다.

## 범위

- 분석 모델, 확률, 좌표 계산 및 데이터 payload는 변경하지 않는다.
- Evidence 패널의 제목과 두 근거 그룹의 제목·부제만 변경한다.
- 현재점은 실물경제 근거, 미래 점선은 실물경제 근거에 전망 맥락을 추가해 계산한다는 기존 모델 경계를 그대로 설명한다.

## 문구

- 패널 제목: `현재와 전망의 판단 근거`
- 현재 그룹 제목: `현재 위치의 근거`
- 현재 그룹 부제: `현재점에 반영되는 생산·소비와 고용·소득`
- 전망 그룹 제목: `1·2개월 전망에 추가되는 근거`
- 전망 그룹 부제: `현재 근거에 더해 미래 확률을 조정하는 금융·선행 여건과 물가·정책`

## 완료 조건

- 기존의 `국면을 움직인 근거`, `실물경제 근거`, `전망 맥락` 문구가 Evidence 패널에서 제거된다.
- 새 문구가 React 원본과 빌드된 정적 컴포넌트에 반영된다.
- 분석 데이터 및 화면 구조에는 회귀가 없다.
