# Economic Cycle Asset Signal Card Copy Design

## 목적

경제사이클 자산 카드가 `경제 배경 우호 / 가격 확인 하락 확인`처럼 추상적인 단어를 나열해 사용자가 의미를 다시 해석해야 하는 문제를 줄인다.

미국 경제 전체의 우세 국면과 자산별 현재 환경을 분리하고, 금·달러는 미국 경기 신호와 실제 시장가격의 관계를 한 문장으로 바로 이해하게 한다.

## 사용자 질문

각 카드에서 다음 세 가지를 한 번에 답한다.

1. 미국 경기지표는 이 자산에 어떤 조건인가?
2. 실제 가격은 어느 방향으로 움직였는가?
3. 두 신호는 같은 방향인가, 다른 방향인가?

## 범위

- 기존 다섯 카드 `채권·금리 / 주식 / 금 / 달러 / 원자재`를 유지한다.
- 경제사이클 확률, publication gate, factor 계산, 가격 수익률 계산 기준은 변경하지 않는다.
- 새 가이드 패널, 외부 provider 호출, 자산 가격 예측, 목표가격, 매매 신호를 추가하지 않는다.
- UI 문구는 현재 read model 값에서 동적으로 생성하며 특정 날짜의 실제 결과를 하드코딩하지 않는다.

## 카드 제목

기존 `경제 국면: 회복`은 모든 자산이 회복 중인 것처럼 읽히므로 카드에서 제거한다.

- 금·달러: `미국 경기 신호와 시장 가격 비교`
- 가격 확인이 없는 자산: `현재 환경: {혼재 | 부담 | 지지 | 자료 부족}`

미국 경제 전체의 우세 국면은 기존 경제사이클 상위 영역에서만 읽는다.

## 금·달러 상단 비교

기존의 추상적인 `경제 배경 / 가격 확인 / 종합 판단`을 다음 세 칸으로 바꾼다.

| 항목 | 금 예시 | 달러 예시 |
|---|---|---|
| 미국 경기 신호 | `금을 지지` | `달러에 부담` |
| 실제 가격 | `하락` | `상승` |
| 두 신호 관계 | `서로 다른 방향` | `서로 다른 방향` |

내부 상태값 `FAVORABLE/BURDEN`, `RISING/FALLING/MIXED`, `ALIGNED/DIVERGENCE`는 유지한다. 화면 문구만 자산과 관계가 드러나는 사용자 언어로 번역한다.

## 설명 문장

### 금 예시

> 미국 경기지표에서는 생산·고용 약화와 물가·정책 압력이 금의 방어 수요를 지지하지만, 실제 가격은 최근 1개월과 3개월 모두 하락해 두 신호가 엇갈립니다.

### 달러 예시

> 미국 경기지표에서는 생산·고용 약화 등 달러 부담 요인이 우세하지만, 실제 달러지수는 최근 1개월과 3개월 상승해 두 신호가 엇갈립니다.

문장은 현재 factor driver와 가격 상태에서 동적으로 만든다. 특정 자산에 대해 확인되지 않은 원인을 추정하지 않는다. 예를 들어 달러 상승을 해외 금리 차이 때문이라고 카드에서 단정하지 않는다.

### 가격 확인이 없는 자산

채권·금리, 주식, 원자재는 표시된 핵심 근거를 자연어로 연결하고 마지막에 현재 환경 판정을 붙인다.

예시:

> 생산·소비 약화는 채권에 지지 요인이지만 물가·정책 압력은 부담으로 작용해, 현재 신호가 한 방향으로 모이지 않습니다.

## 가격 기간 표기

가격 수익률은 달력 기간이 아니라 기존 거래일 기준을 유지하고 화면에서 명시한다.

- `1주(5거래일)`
- `1개월(21거래일)`
- `3개월(63거래일)`

가격 기준일과 symbol을 계속 표시한다. `GC=F` / `DX-Y.NYB` 연속선물에는 계약 교체 효과가 포함될 수 있다는 기존 방법론 경계도 유지한다.

## 하단 문구

`바뀌는 조건`을 `향후 확인 조건`으로 바꾼다. 이는 가격 전망이 아니라 현재 판단을 다시 확인할 조건이다.

## 자료 부족과 혼재

- 가격이 없거나 64개 미만이거나 7일 이상 오래되면 `실제 가격: 확인 대기`, `두 신호 관계: 비교 대기`로 표시한다.
- 미국 경기 신호가 혼재하거나 가격이 혼재하면 `두 신호 관계: 판단 유보`로 표시한다.
- 근거가 부족하면 원인을 설명하고 방향을 만들어내지 않는다.

## 구현 경계

- `finance/economic_cycle_interpretation.py`: 자산별 사용자 문장과 사용자 언어 label 생성
- `app/services/overview/economic_cycle.py`: 기존 payload 전달 및 가격 read failure isolation 유지
- `app/web/streamlit_components/economic_cycle_workbench/src/EconomicCycleWorkbench.tsx`: 새 비교 구조와 문구 표시
- `app/web/streamlit_components/economic_cycle_workbench/src/style.css`: 세 칸 비교와 반응형 배치
- `tests/test_economic_cycle_service.py`: 동적 문장 및 상태별 label 계약
- `tests/test_market_context_economic_cycle.py`: React source contract

## 완료 조건

- 카드마다 미국 경제 전체 국면이 자산 상태처럼 반복되지 않는다.
- 금·달러는 미국 경기 신호, 실제 가격, 두 신호 관계를 별도 항목으로 읽을 수 있다.
- 카드 설명 한 문장만 읽어도 핵심 근거와 실제 가격의 관계를 이해할 수 있다.
- 가격 기간이 5/21/63거래일임을 화면에서 확인할 수 있다.
- 가격 예측이나 원인 추정으로 확대하지 않는다.
- focused Python tests, React build, desktop/mobile overflow 검증을 통과한다.
