# Economic Cycle Rate Unit Copy Design

## 목적

채권·금리 카드의 현재 금리 수준과 변화량 단위를 사용자가 바로 구분하도록 표시 문구만 다듬는다.

## 변경 범위

- 미국 2년·10년 국채 수익률의 현재 값은 `4.13 percent` 대신 `연 4.13%` 형식으로 표시한다.
- 10년-2년 금리차의 현재 값은 기존처럼 `42.00 bp` 형식을 유지한다.
- 채권·금리의 `현재 움직임` 제목 아래에 `현재 값은 최신 저장 관측치이며, 금리 변화는 bp 기준입니다.`를 한 줄로 표시한다.
- 1개월·3개월 변화량의 `bp` 계산, 데이터 기준일, 다른 자산의 가격 변화율 표시는 변경하지 않는다.

## 완료 조건

- 수익률 수준은 `연 n.nn%`, 금리차 수준은 `n.nn bp`로 구분된다.
- 금리 변화량이 bp 기준임을 화면에서 바로 확인할 수 있다.
- 기존 경제사이클 테스트와 Vite build가 통과한다.
