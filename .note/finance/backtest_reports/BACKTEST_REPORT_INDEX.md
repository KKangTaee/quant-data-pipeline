# Backtest Report Index

## 목적

이 문서는 `.note/finance/backtest_reports/` 아래의 결과 리포트 문서를 빠르게 찾기 위한 인덱스다.

## 현재 상태

이 폴더는 새로 만든 결과 리포트 전용 폴더다.

기존 phase 문서에 남아 있는 주요 backtest 결과 문서는 당분간 그대로 두고, 앞으로 새로 만드는 결과 리포트부터 이 폴더 기준으로 누적한다.

## 참고용 기존 대표 문서

아래 문서들은 아직 phase13 폴더에 있지만, 앞으로 이 폴더가 맡게 될 문서 성격의 대표 예시다.

- `../phase13/PHASE13_STRICT_ANNUAL_FAMILY_BACKTEST_SUMMARY.md`
  - `Quality / Value / Quality + Value` strict annual family 백테스트 결과 요약
- `../phase13/PHASE13_STRICT_ANNUAL_COVERAGE300_500_1000_TARGET_SEARCH.md`
  - coverage 확대 탐색 결과
- `../phase13/PHASE13_REAL_MONEY_CANDIDATE_SPY_MDD25_SEARCH.md`
  - `real_money_candidate + SPY 초과 CAGR + MDD 25% 이내` 탐색 결과
- `../phase13/PHASE13_VALUE_RAW_WINNER_BACKTEST_GUIDE.md`
  - strongest raw winner 재현 가이드

## 운영 메모

앞으로 새 report를 만들 때는:

1. 이 폴더에 문서를 만든다
2. 여기 index에 추가한다
3. 필요하면 phase 문서에서 링크한다
