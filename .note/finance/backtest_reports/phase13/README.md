# Phase 13 Backtest Reports

## 목적

이 폴더는 Phase 13 동안 만들어진 결과 중심 backtest Markdown 문서를 모아두는 공간이다.

쉽게 말하면:

- phase13 폴더에는 실행 문서
- 이 폴더에는 결과 문서

를 두는 구조다.

## 포함된 문서 성격

- family summary
- target search
- `SPY` 비교 탐색
- coverage 확장 탐색
- strongest winner 재현 가이드

## 호환성 메모

기존 `.note/finance/phase13/` 경로에는 짧은 안내 stub를 남겨두었다.

즉:

- 예전 경로로 열어도 새 위치를 바로 따라갈 수 있고
- 앞으로는 이 폴더를 canonical result-report 위치로 보면 된다
