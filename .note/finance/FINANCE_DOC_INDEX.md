| м§„н–ү мғҒнғң | мү¬мҡҙ лң» |
|---|---|
| `planned` | кі„нҡҚл§Ң мһҲкі  м•„м§Ғ ліёкІ© мһ‘м—… м „мқҙлӢӨ. |

| `partial_complete` | мқјл¶Җ мһ‘м—… лӢЁмң„л§Ң лҒқлӮ¬лӢӨ. phase м „мІҙ мҷ„лЈҢлҠ” м•„лӢҲлӢӨ. |
| `implementation_complete` | кө¬нҳ„кіј л¬ём„ң 1м°Ё мһ‘м—…мқҖ лҒқлӮ¬лӢӨ. |
| `practical_closeout` | нҳ„мһ¬ лІ”мң„лҠ” лӢӨмқҢ лӢЁкі„лЎң л„ҳкёё мҲҳ мһҲмқ„ л§ҢнҒј м •лҰ¬лҗҳм—ҲлӢӨ. |
