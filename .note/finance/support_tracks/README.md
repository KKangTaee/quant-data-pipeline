# Finance Support Tracks

## 목적

이 폴더는 main finance phase와 직접 같은 줄에 놓기 어려운 보조 작업을 관리한다.

주로 plugin, skill, workflow automation, roadmap rebase 같은 지원성 문서를 둔다.
이 문서들은 제품 기능 phase 자체는 아니지만,
finance 작업을 반복 가능하게 만드는 데 필요한 보조 트랙이다.

## 현재 문서 구분

| 구분 | 대표 문서 |
|---|---|
| Research automation support | `RESEARCH_AUTOMATION_*` 문서 |
| Plugin / skill 검토 | `CODEX_PLUGIN_AND_SKILL_APPLICATION_REVIEW_20260413.md` |
| Roadmap rebase 논의 | `ROADMAP_REBASE_PHASE18_TO_PHASE25_20260414.md` |

## 관리 기준

- 제품 기능 phase 문서는 `../phase*/`에 둔다.
- active roadmap은 `../MASTER_PHASE_ROADMAP.md`가 기준이다.
- 이 폴더에는 phase 밖에서 작업 효율, automation, planning support에 영향을 주는 문서만 둔다.
