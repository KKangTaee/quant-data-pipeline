# Finance Saved Artifacts

Status: Active
Last Verified: 2026-05-28

이 폴더는 사용자가 명시적으로 저장한 재사용 가능 설정 JSONL을 둔다.

## 포함 파일

- `SAVED_PORTFOLIOS.jsonl`: Compare / Portfolio Builder에서 저장한 weighted portfolio 설정
- `SAVED_PORTFOLIO_MIXES.jsonl`: Portfolio Selection V2에서 저장한 reusable mix setup

## 사용 기준

saved 파일은 registry와 다르게 후보 승인 기록이 아니다. 사용자가 같은 설정을 다시 열거나 replay하기 위한 저장소다.

saved setup은 Practical Validation result나 Final Review decision을 대체하지 않는다.
새 saved 파일을 추가하기 전에는 `.aiworkspace/note/finance/docs/data/STORAGE_GOVERNANCE.md`의 checklist를 확인한다.
