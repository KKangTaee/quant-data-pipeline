# 2026 Strategy Search Reports

Status: Active
Last Verified: 2026-05-12

이 폴더는 legacy phase13~phase18에서 생성된 전략 탐색, 후보 개선, 조건 변경, representative rerun report를 보관한다.

이 문서들은 현재 후보 source-of-truth가 아니라, 과거에 어떤 조건을 탐색했고 왜 현재 전략 hub/log 해석으로 이어졌는지를 추적하기 위한 원본성 report다.

## How To Use

- 전략 family의 현재 해석은 먼저 `../../../strategies/` hub와 log에서 본다.
- 왜 그런 후보가 생겼는지 원문 흐름이 필요할 때 이 폴더를 연다.
- 새 분석 report는 이 폴더가 아니라 해당 연도의 목적별 위치에서 시작한다.
