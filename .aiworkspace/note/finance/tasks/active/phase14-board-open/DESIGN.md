## Design

Phase 14 is a prioritization phase, not an implementation phase.
The board starts from Phase 13's carry-forward matrix and narrows the second-cycle entry point.

Initial task split:

- 14-0 board open
- 14-1 candidate prioritization matrix
- 14-2 first implementation slice design
- 14-3 handoff closeout

The initial hypothesis favors immediately implementable workflow hardening before source-dependent or broker-grade design work, but 14-1 owns the final ranking.
