# Plan

## 이걸 하는 이유?

Overview 상단에서 미국 정규장 기준일, 개장 / 종료 시간, 휴장 사유를 바로 확인할 수 있어야 Market Movers 자동 갱신이나 장중 데이터 해석 시 현재 시장 세션 맥락을 빠르게 판단할 수 있다.

## Scope

- Overview 상단에 미국장 세션 배너를 추가한다.
- 미국 동부시간 기준 NYSE 정규장 일정을 계산하고, 표시에는 ET / KST 시간을 함께 제공한다.
- 주말과 NYSE 주요 휴장일은 휴장 사유를 표시한다.
- 외부 API나 신규 유료 데이터 소스는 사용하지 않는다.

## Acceptance

- 거래일이면 Open / Close 시간이 보인다.
- 휴장일이면 `휴장` 상태와 사유, 다음 개장 / 종료 시간이 보인다.
- Overview 브라우저 화면에서 배너가 깨지지 않는다.
