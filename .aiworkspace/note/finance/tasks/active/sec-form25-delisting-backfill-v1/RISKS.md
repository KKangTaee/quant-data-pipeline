# SEC Form 25 Delisting Backfill V1 Risks

- SEC ticker mapping can miss old or changed tickers; unmapped symbols must remain explicit gaps.
- Form 25 confirms delisting / withdrawal evidence but does not create complete historical universe membership.
- Absence of Form 25 must not be interpreted as active listing proof.
- SEC scripted access requires a declared user agent and request-rate discipline.
- Because Form 25 does not provide first listing date, these rows can be actual delisting evidence while still insufficient for full requested-period membership coverage by themselves.
