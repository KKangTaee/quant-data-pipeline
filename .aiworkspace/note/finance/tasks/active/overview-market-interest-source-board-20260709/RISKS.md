# Risks

- MarketWatch/WSJ/Nasdaq 공개 페이지는 약관, paywall, 동적 렌더링, 제3자 데이터 라이선스 이슈가 있어 이번 차수에서는 자동 수집하지 않는다.
- yfinance는 비공식 Yahoo 메타데이터 접근이므로 세션 전용 조사 단서로만 표시하고 durable storage로 승격하지 않는다.
- 이번 변경은 시각적 구분과 source policy 표시 개선이다. 여러 사이트의 analyst value를 자동 파싱 / 정규화하는 기능은 별도 승인 후 provider policy 검토가 필요하다.
