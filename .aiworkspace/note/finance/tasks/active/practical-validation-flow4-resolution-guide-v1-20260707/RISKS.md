# Risks

- `resolution_guide`는 user-facing card contract이며 gate severity를 바꾸지 않는다. 후속 작업에서 copy를 조정할 때 `NEEDS_INPUT` / `REVIEW` 의미를 바꾸지 않도록 주의한다.
- 일부 내부 evidence appendix와 board registry label은 `Data Coverage Audit`, `Validation Efficacy Audit` 같은 기술 taxonomy를 유지한다. 이 이름은 user-facing Flow4 위치 안내와 구분한다.
