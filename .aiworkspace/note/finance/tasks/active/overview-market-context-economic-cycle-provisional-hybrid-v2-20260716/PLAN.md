# Economic Cycle Provisional Hybrid V2 Plan

Status: In Progress
Last Updated: 2026-07-16

## 이걸 하는 이유?

계산 가능한 경제사이클 결과를 validation gate 미달이라는 이유만으로 모두 숨기면 사용자가 현재 위치와 미래 경로를 읽을 수 없다. 검증 기준은 유지하면서 잠정 추정과 검증 완료를 분리하고, 처음 승인한 2×2 혼합형으로 사용 가치를 회복한다.

## Roadmap

1. 기존 명세·코드·데이터 계약 확인
2. 잠정/검증/판단불가 상태 계약과 TDD
3. 2×2 혼합형 확률 경로와 ribbon 구현
4. actual snapshot 갱신과 Browser QA
5. 문서 동기화·최종 검증·커밋
