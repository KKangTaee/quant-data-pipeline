# Risks

- Live DB contents are environment-dependent. Contract tests should validate read-model behavior with injected evidence where needed.
- Full visual Streamlit layout QA is not equivalent to service contract QA; run browser/UI QA separately if layout changes resume.
- Remaining future work should be opened as a new phase/task instead of extending P3 indefinitely.
- P3 closeout does not mean the product is fully live-investment ready. It means selected monitoring read models are connected and storage boundaries are preserved.
