# Economic Cycle Asset Pathways Stages 3-5 V1 Plan

## 이걸 하는 이유?

금·달러 파일럿에서 확립한 측정 경로 원칙을 채권·금리 구조, S&P 500, WTI·구리·금으로 확장해 자산가격을 미국 경제상태의 단순 정답으로 취급하지 않기 위해서다.

## Scope

- 3차: 2년물·10년물·10년-2년 금리차와 10년 실질금리·기대인플레이션
- 4차: S&P 500 가격·실질금리·신용스프레드·VIX·actual EPS
- 5차: WTI·구리·금과 측정 가능한 EIA 수급·달러·활동 경로
- 공통 관측 UI와 actual data/browser QA

## Stop Condition

3·4·5차가 `PATHWAYS_NOT_CONNECTED` 없이 실제 저장 데이터를 표시하고, 인과·확률·매매 문구 없이 focused tests·React build·browser QA를 통과하면 종료한다.
