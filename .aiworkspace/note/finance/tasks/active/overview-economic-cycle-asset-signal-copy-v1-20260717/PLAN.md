# Economic Cycle Asset Signal Copy V1 Plan

## 이걸 하는 이유?

`경제 국면: 회복`, `경제 배경 우호`, `가격 확인 하락 확인`처럼 서로 다른 의미가 한 카드에 섞여 자산 자체가 회복 중인 것처럼 읽히는 문제를 줄인다.

1. 자산별 미국 경기 신호와 동적 설명 문장
2. 실제 가격과 두 신호 관계를 분리한 React 비교 카드
3. 5/21/63거래일 표기, actual read model, desktop/420px QA와 문서 정렬

Stop condition: 미국 경제 전체 국면은 상위 영역에만 남고, 금·달러 카드에서 `미국 경기 신호 / 실제 가격 / 두 신호 관계`와 정확한 거래일 기간을 바로 읽을 수 있다.
