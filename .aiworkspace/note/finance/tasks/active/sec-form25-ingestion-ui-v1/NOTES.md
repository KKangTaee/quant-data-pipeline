# SEC Form 25 Ingestion UI V1 Notes

- This UI should make the collector reachable but should not imply complete historical universe coverage.
- The UI copy must state that Form 25 absence is not active-listing proof.
- SEC scripted access should surface the `SEC_USER_AGENT` requirement without storing a user-specific credential or profile.
- The tab uses the existing ingestion scheduler and result display. It does not add a separate registry, saved setup, user memo, or report export.
