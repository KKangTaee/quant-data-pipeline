# Economic Cycle Asset Context V1 Notes

- 기존 섹션은 evidence_count만 포함한 정적 설명이어서 네 카드가 실질적으로 구분되지 않았다.
- 실제 snapshot은 실물·고용 약화, 금융·선행·물가/정책 강화 조합이다.
- 사용자가 정리한 학습 자료의 핵심인 실물, 금리/신용, 물가/정책의 교차 확인을 read model에 반영한다.
- `혼재` 카드는 강한 우호 근거와 강한 부담 근거를 하나씩 보여줘 충돌 이유를 숨기지 않는다.
- `금·달러`는 두 가격의 동일 방향 예측이 아니라 방어 수요·실질금리·정책 불확실성 맥락으로 제한한다.
