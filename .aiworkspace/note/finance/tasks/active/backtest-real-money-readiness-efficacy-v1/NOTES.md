# Backtest Real-Money Readiness Efficacy V1 Notes

## Decision

- Backtest Real-Money remains a first-pass candidate readiness filter.
- `Suggested Route` is guidance, not evidence.
- `Execution Preview` should summarize execution / validation burden sources, not later-stage probation or monitoring state.
- Cost / turnover values are useful only when their estimation status is visible.
- Current legacy `out_of_sample_review_status` is displayed as a simple `Split-Period Check`, not formal walk-forward / holdout validation.
