# Risks

- Catalog status는 source confidence context이며, provider quality를 정량 보증하지 않는다.
- Free futures / earnings estimate source caveat는 그대로 노출하지만 provider replacement는 후속 정책 결정이 필요하다.
- 현재 catalog는 cockpit snapshot 재사용형 first pass라 Reference companion, durable source table, provider score persistence는 포함하지 않는다.
- 실제 source thresholds는 기존 Data Health / event / sentiment / futures read model의 semantics를 따른다. 4차는 threshold를 새로 정의하지 않는다.
