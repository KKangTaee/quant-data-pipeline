# RISKS - Investability Evidence Packet V1

Status: Active
Last Updated: 2026-05-28

## Risks

| Risk | Mitigation |
| --- | --- |
| Final Review becomes too noisy | Use compact packet summary and expandable/detail tables already present in the screen. |
| Gate becomes too strict while provider coverage is partial | V1 blocks only selected route; hold / reject / re-review remain available. |
| User memo storage grows again | Do not add new memo fields beyond existing operator reason / constraints / next action. |
| JSONL row grows too much | Store compact packet snapshot, not full raw provider / holdings / macro data. |
