# Risks

- React custom component can blank silently if build asset paths are wrong; tests should check generated `index.html` asset paths.
- Moving actual controls into React would require bidirectional Streamlit state/event handling. This task keeps the new panel read-only.
- Strict annual and strict quarterly variants do not expose identical policy/guardrail capabilities; the read model must not imply quarterly prototype has the annual promotion policy gate.

## Residual Notes
